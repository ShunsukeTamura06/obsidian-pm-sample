## 2. 進行中案件

```dataview
TABLE file.link AS 案件, 担当, 優先度, 期限, 工数進捗 + "%" AS 進捗

WHERE ステータス = "進行中"
SORT 優先度 DESC, 期限 ASC
```

## 3. 今週完了・レビュー中

```dataview
TABLE file.link AS 案件, 担当, ステータス, 期限
FROM "01_案件"
WHERE ステータス = "レビュー中" OR ステータス = "完了"
SORT ステータス ASC
```

## 4. 要対応事項

```dataview
TABLE file.link AS 案件, 担当, 期限, 優先度
FROM "01_案件"
WHERE ステータス != "完了" AND date(期限) <= date(today) + dur(14 days)
SORT 期限 ASC
```

## 5. 担当者別ロード

```dataview
TABLE WITHOUT ID 担当 AS 担当者, length(rows) AS 抱える案件数, length(filter(rows.ステータス,(s)=>s="進行中")) AS うち進行中
FROM "01_案件"
WHERE ステータス != "完了"
GROUP BY 担当
SORT length(rows) DESC
```
