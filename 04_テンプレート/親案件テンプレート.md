## タスクノート一覧

```dataview
TABLE
    file.link AS タスク,
    担当,
    ステータス,
    期限,
    工数進捗 + "%" AS 進捗
FROM "01_案件/{{title}}"
WHERE file.name != this.file.name
SORT ステータス ASC, 期限 ASC
```

## タスク集計

```dataview
TABLE WITHOUT ID
    length(rows) AS 総タスク数,
    length(filter(rows.ステータス,(s)=>s="完了")) AS 完了,
    length(filter(rows.ステータス,(s)=>s="進行中")) AS 進行中,
    length(filter(rows.ステータス,(s)=>s="未着手")) AS 未着手,
    round(sum(rows.工数進捗)/length(rows)) + "%" AS 平均進捗
FROM "01_案件/{{title}}"
WHERE file.name != this.file.name
GROUP BY true
```
