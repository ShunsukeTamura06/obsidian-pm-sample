## 進行中案件

```dataview
TABLE file.link AS 案件, 担当, 優先度, 期限, 工数進捗 + "%" AS 進捗
FROM "01_案件"
WHERE ステータス = "進行中" AND 案件 = null
SORT 優先度 DESC, 期限 ASC
```

## 要対応（期限14日以内かつ未完了）

```dataview
TABLE file.link AS 案件, 担当, 期限
FROM "01_案件"
WHERE ステータス != "完了" AND date(期限) <= date(today) + dur(14 days)
SORT 期限 ASC
```

## 担当者別ロード

```dataview
TABLE WITHOUT ID 担当 AS 担当者, length(rows) AS 抱える案件数
FROM "01_案件"
WHERE ステータス != "完了" AND 案件 = null
GROUP BY 担当
SORT length(rows) DESC
```
