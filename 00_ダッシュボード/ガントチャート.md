# ガントチャート

%% 用語: 案件（MOCノート）→ サブタスク（案件内.mdノート）→ アクション（チェックボックス項目） %%

## サブタスクガント

```dataviewjs
const isMOC = p => p.file.folder.split("/").pop() === p.file.name;
const subtasks = dv.pages('"01_案件"')
    .where(p => p.file.folder !== "01_案件" && !isMOC(p) && p.開始日 && p.期限)
    .sort(p => p.開始日);

const statusMap = {"完了":"done", "進行中":"active", "保留":"crit", "未着手":""};
let g = "gantt\n    title サブタスクガント\n    dateFormat YYYY-MM-DD\n    axisFormat %m/%d\n";

const grouped = {};
for (const t of subtasks) {
    const key = t.file.folder.split("/").pop();
    if (!grouped[key]) grouped[key] = [];
    grouped[key].push(t);
}
for (const [sec, items] of Object.entries(grouped)) {
    g += `    section ${sec}\n`;
    for (const t of items) {
        const st = statusMap[t.ステータス] || "";
        const s = t.開始日.toISODate ? t.開始日.toISODate() : t.開始日;
        const e = t.期限.toISODate ? t.期限.toISODate() : t.期限;
        g += `    ${t.file.name} :${st ? st + ", " : ""}${s}, ${e}\n`;
    }
}
await dv.paragraph("```mermaid\n" + g + "\n```");
```

> **凡例**: 青＝進行中 / 緑＝完了 / 赤＝保留 / グレー＝未着手

## 案件別サブタスク・アクション一覧

```dataviewjs
const isMOC = p => p.file.folder.split("/").pop() === p.file.name;
const projects = dv.pages('"01_案件"')
    .where(p => p.file.folder !== "01_案件" && isMOC(p))
    .sort(p => p.file.name);

for (const proj of projects) {
    dv.header(3, proj.file.link);

    const subtasks = dv.pages(`"${proj.file.folder}"`)
        .where(p => p.file.name !== proj.file.name)
        .sort(p => p.開始日);

    if (subtasks.length === 0) {
        dv.paragraph("サブタスクなし");
        continue;
    }

    for (const st of subtasks) {
        // サブタスクのヘッダー行
        const progress = st.工数進捗 != null ? ` ${st.工数進捗}%` : "";
        const badge = `\`${st.ステータス ?? "-"}${progress}\``;
        dv.header(4, `${st.file.link}　${badge}`);

        // 概要
        if (st.概要) dv.paragraph(`> ${st.概要}`);

        // アクション（チェックボックス）を未完了・完了に分けて表示
        const actions = st.file.tasks;
        if (actions.length === 0) continue;

        const incomplete = actions.filter(a => !a.completed);
        const complete   = actions.filter(a => a.completed);

        if (incomplete.length > 0) {
            dv.paragraph("**未完了アクション**");
            dv.taskList(incomplete, false);
        }
        if (complete.length > 0) {
            dv.paragraph("**完了済みアクション**");
            dv.taskList(complete, false);
        }
    }
}
```
