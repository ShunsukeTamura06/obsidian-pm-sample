# 全案件一覧

%% 判定ロジック：フォルダ名 == ファイル名 → MOC（案件）、それ以外 → サブタスク %%

## 案件一覧

```dataviewjs
const fmt = d => d?.toFormat ? d.toFormat("yyyy-MM-dd") : (d ?? "-");
const isMOC = p => p.file.folder.split("/").pop() === p.file.name;
const projects = dv.pages('"01_案件"')
    .where(p => p.file.folder !== "01_案件" && isMOC(p))
    .sort(p => p.優先度, "desc");
dv.table(["案件", "担当", "ステータス", "優先度", "期限"],
    projects.map(p => [p.file.link, p.担当, p.ステータス, p.優先度, fmt(p.期限)])
);
```

## 案件別サブタスク集計

```dataviewjs
const fmt = d => d?.toFormat ? d.toFormat("yyyy-MM-dd") : (d ?? "-");
const isMOC = p => p.file.folder.split("/").pop() === p.file.name;
const isTask = p => p.file.folder !== "01_案件" && !isMOC(p);

const projects = dv.pages('"01_案件"').where(p => p.file.folder !== "01_案件" && isMOC(p));

const rows = [];
for (const proj of projects) {
    const tasks = dv.pages(`"${proj.file.folder}"`).where(p => isTask(p));
    const total = tasks.length;
    if (total === 0) { rows.push([proj.file.link, 0, "-", "-", "-", "-"]); continue; }
    const done = tasks.filter(t => t.ステータス === "完了").length;
    const inp  = tasks.filter(t => t.ステータス === "進行中").length;
    const ns   = tasks.filter(t => t.ステータス === "未着手").length;
    const avg  = Math.round(tasks.array().reduce((s, t) => s + (t.工数進捗 || 0), 0) / total);
    rows.push([proj.file.link, total, done, inp, ns, avg + "%"]);
}
dv.table(["案件", "サブタスク数", "完了", "進行中", "未着手", "平均進捗"], rows);
```

## 要対応（期限14日以内かつ未完了）

```dataviewjs
const fmt = d => d?.toFormat ? d.toFormat("yyyy-MM-dd") : (d ?? "-");
const isMOC = p => p.file.folder.split("/").pop() === p.file.name;
const pages = dv.pages('"01_案件"')
    .where(p => p.file.folder !== "01_案件"
        && p.ステータス !== "完了"
        && p.期限
        && dv.date(p.期限) <= dv.date("today").plus({days: 14}))
    .sort(p => p.期限);
dv.table(["ノート", "種別", "担当", "期限"],
    pages.map(p => [p.file.link, isMOC(p) ? "案件" : "サブタスク", p.担当, fmt(p.期限)])
);
```
