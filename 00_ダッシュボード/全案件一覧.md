# 全案件一覧

このノートはすべての案件を横断的に把握するためのダッシュボードです。

## 親案件一覧

```dataview
TABLE file.link AS 親案件, 担当, 案件種別, 優先度, 期限
FROM "01_案件"
WHERE 案件レベル = "親"
SORT 優先度 DESC, 期限 ASC
```

## 親案件の進捗サマリ（子の集計）

```dataviewjs
const parents = dv.pages('"01_案件"').where(p => p["案件レベル"] === "親");
const rows = [];
for (const parent of parents) {
    const children = dv.pages('"01_案件"').where(c => c["親案件"] && c["親案件"].path === parent.file.path);
    if (children.length === 0) { rows.push([parent.file.link, 0, 0, 0, 0, "—"]); continue; }
    const total = children.length;
    const done = children.filter(c => c["ステータス"] === "完了").length;
    const inp = children.filter(c => c["ステータス"] === "進行中").length;
    const ns = children.filter(c => c["ステータス"] === "未着手").length;
    const avg = Math.round(children.array().reduce((s, c) => s + (c["工数進捗"] || 0), 0) / total);
    rows.push([parent.file.link, total, done, inp, ns, avg + "%"]);
}
dv.table(["親案件", "子案件数", "完了", "進行中", "未着手", "平均進捗"], rows);
```

## 進行中案件（親・単体）

```dataview
TABLE file.link AS 案件, 担当, 優先度, 期限, 工数進捗 + "%" AS 進捗
FROM "01_案件"
WHERE ステータス = "進行中" AND (案件レベル = "単体" OR 案件レベル = "親")
SORT 優先度 DESC, 期限 ASC
```

## 子案件一覧

```dataview
TABLE file.link AS 子案件, 親案件, 担当, ステータス, 期限, 工数進捗 + "%" AS 進捗
FROM "01_案件"
WHERE 案件レベル = "子"
SORT 親案件 ASC, 期限 ASC
```

## 要注意（期限14日以内かつ未完了）

```dataview
TABLE file.link AS 案件, 案件レベル, 担当, ステータス, 期限
FROM "01_案件"
WHERE ステータス != "完了" AND date(期限) <= date(today) + dur(14 days)
SORT 期限 ASC
```

## レビュー待ち

```dataview
TABLE file.link AS 案件, 担当, 期限, 工数進捗 + "%" AS 進捗
FROM "01_案件"
WHERE ステータス = "レビュー中"
SORT 期限 ASC
```

## ステータス別サマリ

```dataview
TABLE WITHOUT ID ステータス, length(rows) AS 件数
FROM "01_案件"
WHERE 案件レベル != "親"
GROUP BY ステータス
```
