# 担当者別ロード

## 担当者別タスク数

```dataviewjs
const isMOC = p => p.file.folder.split("/").pop() === p.file.name;
const tasks = dv.pages('"01_案件"')
    .where(p => p.file.folder !== "01_案件" && !isMOC(p) && p.ステータス !== "完了");

const byPerson = {};
for (const t of tasks) {
    const name = t.担当 || "未割当";
    if (!byPerson[name]) byPerson[name] = {total:0, inp:0, ns:0};
    byPerson[name].total++;
    if (t.ステータス === "進行中") byPerson[name].inp++;
    if (t.ステータス === "未着手") byPerson[name].ns++;
}

const rows = Object.entries(byPerson)
    .sort((a,b) => b[1].total - a[1].total)
    .map(([name, v]) => [name, v.total, v.inp, v.ns]);
dv.table(["担当者", "引き受けタスク", "進行中", "未着手"], rows);
```

## 担当者別タスク一覧

```dataviewjs
const isMOC = p => p.file.folder.split("/").pop() === p.file.name;
const tasks = dv.pages('"01_案件"')
    .where(p => p.file.folder !== "01_案件" && !isMOC(p) && p.ステータス !== "完了")
    .sort(p => [p.担当, p.期限]);
dv.table(["タスク", "案件", "担当", "ステータス", "期限"],
    tasks.map(t => [
        t.file.link,
        t.file.folder.split("/").pop(),
        t.担当, t.ステータス, t.期限
    ])
);
```
