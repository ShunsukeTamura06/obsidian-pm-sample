# 担当者別ロード

## 担当者別タスク数

```dataview
TABLE WITHOUT ID
    担当 AS 担当者,
    length(rows) AS 引き受けタスク,
    length(filter(rows.ステータス, (s) => s = "進行中")) AS 進行中,
    length(filter(rows.ステータス, (s) => s = "未着手")) AS 未着手
FROM "01_案件"
WHERE 種別 = "タスク" AND ステータス != "完了"
GROUP BY 担当
SORT length(rows) DESC
```

## 担当者別タスク一覧

```dataview
TABLE file.link AS タスク, 案件, ステータス, 期限
FROM "01_案件"
WHERE 種別 = "タスク" AND ステータス != "完了"
SORT 担当 ASC, 期限 ASC
```
