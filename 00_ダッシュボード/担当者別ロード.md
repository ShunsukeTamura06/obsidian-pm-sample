# 担当者別ロード

## 担当者別サマリ

```dataview
TABLE WITHOUT ID
    担当 AS 担当者,
    length(rows) AS 総案件数,
    length(filter(rows.ステータス, (s) => s = "進行中")) AS 進行中,
    length(filter(rows.ステータス, (s) => s = "レビュー中")) AS レビュー中,
    length(filter(rows.ステータス, (s) => s = "未着手")) AS 未着手,
    length(filter(rows.ステータス, (s) => s = "完了")) AS 完了
FROM "01_案件"
GROUP BY 担当
SORT length(rows) DESC
```

## 高優先度案件の担当分布

```dataview
TABLE file.link AS 案件, 担当, ステータス, 期限
FROM "01_案件"
WHERE 優先度 = "高" AND ステータス != "完了"
SORT 担当 ASC, 期限 ASC
```
