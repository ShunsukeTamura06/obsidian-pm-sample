## 概要

レガシーDBを統合し、全社共通データ基盤を構築する案件群。BIツール連携とデータガバナンス整備を含む。

## 子案件一覧

```dataview
TABLE file.link AS 子案件, 担当, ステータス, 期限, 工数進捗 + "%" AS 進捗
FROM "01_案件/データ基盤刷新案件"
WHERE 親案件 = this.file.link
SORT 期限 ASC
```

## 子案件の集計

```dataview
TABLE WITHOUT ID length(rows) AS 子案件数, length(filter(rows.ステータス,(s)=>s="完了")) AS 完了, length(filter(rows.ステータス,(s)=>s="進行中")) AS 進行中, length(filter(rows.ステータス,(s)=>s="未着手")) AS 未着手, round(sum(rows.工数進捗)/length(rows))+"%" AS 平均進捗
FROM "01_案件/データ基盤刷新案件"
WHERE 親案件 = this.file.link
GROUP BY true
```
